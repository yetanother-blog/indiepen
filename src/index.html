Why Indiepen?

This project was initially developed because we wanted to show code examples on yetanother.blog. In the beginning, we used CodePen and it worked like a charm. But during the usage, we discovered that CodePen is using cookies for their embedded version. Since we try to protect the privacy of our visitors and we were too lazy to implement one of those evil cookie law banners. We decided to build Indiepen.

The idea of this project is to convince CodePen to not set cookies and track our visitors. So we can use CodePen again and get rid of Indiepen because we love CodePen and appreciate their hard work and creativity.

Get started

  1. Provide your code example

    Before we start. We want to let you know that you can only create an embed code example with HyperText Markup Language (HTML), Cascading Style Sheets (CSS) and Javascript. The other thing is that we follow a certain naming convention to scrape the code. Therefore you have to name your files like: 

    
├── index.html
├── main.js
└── styles.css

    Now it’s time to host your code example. There a plenty of options out there. We decided to use GitHub Pages, because it’s the most convenient way for us to create a quick example. Of course, you can also use Netlify, Vercel or other services to host your example. By the end of the day we only need a valid Uniform Resource Locator (URL) that serves a valid code example and by valid we mean it’s HTML, CSS and Javascript. Your URL could look something like that:

    https://examples.yetanother.blog/material-design-tab/

    Tip: Additional support for Github Pages

    You can take a look at our examples repo on GitHub and checkout the documentation of GitHub Pages. Happy Coding!

  2. Copy your code example URL

    Guess what? Now we have to paste this URL into the input field and hit the create example button. This will generate an iframe and put the result into an textarea field below.

  3. Copy your generated code example

    Copy your generated iframe into your website and be happy.
